# STACK.md - Smart College Complaint System

## Frontend Stack
- React.js 18
- React Router v6
- Tailwind CSS v3
- Axios
- Chart.js & react-chartjs-2
- react-hot-toast
- date-fns
- lucide-react / heroicons (assumed for icons)

## Backend Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- bcryptjs
- jsonwebtoken (JWT)
- multer
- cors
- helmet
- express-rate-limit
- express-async-errors

## Tooling & Execution
- npm (package manager)
- concurrently (for running dev servers)
- nodemon (backend auto-restart)
